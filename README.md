# API сервиса YaMDb
### Описание. 

Бэкенд для проекта YaMDb, который собирает отзывы пользователей на произведения. Технологии: Python 3, Django 2.2.19, SQLite3, DRF, Simple-JWT, Git.

### Как запустить проект.

Клонировать репозиторий и перейти в него в командной строке:

```
git clone <cсылка_на_проект_в_git>api_yamdb.git

cd api_yamdb
```

Cоздать и активировать виртуальное окружение:

```
python3 -m venv env
```

```
source env/bin/activate
```

Установить зависимости из файла requirements.txt:

```
python3 -m pip install --upgrade pip

pip install -r requirements.txt
```

Выполнить миграции:

```
python3  api_yamdb/manage.py migrate
```

Запустить проект:

```
python3 api_yamdb/manage.py runserver
```

### Документация к API

Доступна по адресу http://127.0.0.1:8000/redoc/ . Здесь можно увидеть все возможные запросы к api и ответы.

### Ресурсы API YaMDb

* auth: аутентификация.
* users: пользователи.
* titles: произведения, к которым пишут отзывы (определённый фильм, книга или песенка).
* categories: категории (типы) произведений («Фильмы», «Книги», «Музыка»).
* genres: жанры произведений. Одно произведение может быть привязано к нескольким жанрам.
* reviews: отзывы на произведения. Отзыв привязан к определённому произведению.
* comments: комментарии к отзывам. Комментарий привязан к определённому отзыву.
